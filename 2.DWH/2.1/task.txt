1. 
Сущности:
	1) Читатели
	2) Обращения
	3) Книги
	4) Издательство книги
	5) Издательства

2-3. 
Читатели - обращения (один ко многим)
Книги - обращения (один ко многим)
Сущность "Обращения" является связующей между "Читатели" и "Книги".
Каждый читатель имеет уникальный номер читательского билета с помощью которого он может обращаться в библитеку для того-что взять несколько книг. В свою очередь у каждой книги есть определённое количество экземпляров с одинаковым шифром книги, поэтому один шифр могут использовать несколько раз в обращении.

Книги - Издательство книги (один ко многим)
Издательства - Издательство книги (один ко многим)
Сущность "Издательство книги" является связующей между "Издательства" и "Книги".
У каждой книги может быть несколько издательств, а у каждого издательства несколько книг

4. 
PK - Номер читательского билета(Читатели), FK - Номер читательского билета(Обращения).
PK - Шифр книги(Книги), FK - Шифр книги(Обращения).
PK - Шифр книги(Книги), FK - Шифр книги(Издательство книги).
PK - Название(Издательства), FK - Издательство(Издательство книги).
PK - ID_обращения(Обращения).
PK - ID(Издательство книги).
